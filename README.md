# API	Data Coverage	Notes / Free Limits	URL

Yahoo Finance / yfinance Python library	Historical daily stock prices, dividends, splits, and basic indicators	Free, easy to use in Python. Limited real-time data (delayed ~15–20 min).	yfinance

Alpha Vantage	Historical daily, weekly, intraday; fundamental data	Free tier: 5 calls/min, 500 calls/day. JSON or CSV.	Alpha Vantage

Finnhub	Historical prices, real-time quotes, news, earnings	Free: 60 calls/min. Covers multiple exchanges.	Finnhub

Tiingo	Historical prices, news, metadata	Free tier: 500 tickers/month, needs API key.	Tiingo
